# Como publicar o README no GitHub

O README de perfil aparece na página https://github.com/devdezin quando alguém visita seu perfil.

## Passo a passo

### 1. Criar o repositório especial

1. Acesse https://github.com/new
2. **Repository name:** `devdezin` (igual ao seu username)
3. Marque **Public**
4. Marque **Add a README file**
5. Clique em **Create repository**

> O GitHub só exibe README de perfil se o repo tiver o **mesmo nome do usuário**.

### 2. Substituir o README

1. Abra o arquivo `README.md` dentro do repo `devdezin/devdezin`
2. Clique em **Edit** (lápis)
3. Apague tudo e cole o conteúdo de `content/github-profile/README.md` deste projeto
4. Clique em **Commit changes**

### 3. Configurar o perfil

Em https://github.com/settings/profile:

| Campo | Valor |
|-------|--------|
| **Name** | DevDezin |
| **Bio** | Desenvolvedor Full Stack \| React, Next.js, TypeScript \| Sistemas web, PIX e VPS |
| **Company** | DevDezin |
| **Location** | Franca, SP, Brasil |
| **Website** | https://devdezin.com.br |
| **Social accounts → LinkedIn** | https://www.linkedin.com/in/ricardo-ferreira-09966a417/ |

### 4. Fixar repositórios (quando tiver)

Quando subir projetos no GitHub:

1. Vá em https://github.com/devdezin?tab=repositories
2. Abra um repo → **Settings** → marque como favorito ou use **Pin repository** no perfil (até 6 repos fixados)

---

Pronto. Em alguns segundos o README aparece no topo do seu perfil.
